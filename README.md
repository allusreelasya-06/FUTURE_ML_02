# 🎫 Support Ticket Classification System
### Future Interns - Machine Learning Internship | Task 2

## 📌 Project Overview
An ML system that automatically classifies customer support tickets into categories and assigns priority levels.

## 🛠️ Tools & Technologies
- Python
- Scikit-learn
- NLTK
- Pandas & NumPy
- Matplotlib & Seaborn
- Jupyter Notebook

## 📊 Features
- Text cleaning & preprocessing
- Ticket category classification (Technical, Billing, Account)
- Priority tagging (High, Medium, Low)
- Model performance evaluation
- Data visualization

## 📈 Model Performance
- Algorithm: Naive Bayes
- Accuracy: 67%
- Dataset: 15 support tickets
